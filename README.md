# Python Bootcamp – Variables, Input, and Conditions

This repository contains my beginner Python practice code covering **basic programming concepts** such as variables, user input, data types, and conditional statements.

## Topics Covered
* Variables in Python
* Data types (`int`, `float`, `string`, `boolean`)
* Taking input from the user
* Type conversion using `int()`
* Conditional statements (`if`, `elif`, `else`)
* Basic comparison operations

## Part A – Variables and Input
In this section, the program demonstrates:
* Creating and printing variables
* Working with different data types
* Checking the data type using `type()`
* Taking input from the user
  
Examples included in the code:
* Storing a **name**
* Storing **age and height**
* Checking the **type of a variable**
* Using a **boolean variable**
* Taking a number as input from the user
 

## Part B – Conditional Statements
This section demonstrates how to use **if–else conditions** in Python.

The program includes examples such as:
* Adding two numbers entered by the user
* Checking which number is greater
* Checking whether a number is **even or odd**
* Determining whether a student **passes or fails**
* Comparing a number with **10**
These exercises help understand how decisions are made in a Python program.


## Mini Project – Positive or Negative Number Checker
The mini project asks the user to enter a number and determines whether the number is:
* **Positive**
* **Negative**

This project demonstrates how conditional statements work with user input.
Example:
Enter a number: 5
The number is Positive

Enter a number: -3
The number is Negative
